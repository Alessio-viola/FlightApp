1. API OAuth 2.0
        L'API OAuth 2.0 di Google consente alle applicazioni di ottenere l'autorizzazione per accedere alle risorse 
        dell'utente su diversi servizi di Google senza richiedere le credenziali dell'utente. L'utilizzo dell'API 
        OAuth 2.0 di Google consente alle applicazioni di accedere in modo sicuro alle risorse dell'utente su 
        servizi Google specifici senza richiedere le credenziali dell'utente, come applicato nella sezione di login 
        della nostra web application. Ciò offre un'esperienza utente più sicura e consente alle applicazioni di 
        integrarsi in modo più efficace con l'ecosistema di Google.


2. API AMADEUS
        Amadeus è una delle principali società globali di tecnologia per l'industria del viaggio e del turismo. 
        L'API Amadeus offre una vasta gamma di servizi e dati per facilitare lo sviluppo di applicazioni e 
        soluzioni nel settore del viaggio.Nello specifico abbiamo selezionato la seguente funzionalità:

             Prenotazioni di voli: l'API Amadeus consente di accedere a funzionalità di prenotazione di voli, 
             inclusa la ricerca di voli disponibili, il recupero dei dettagli sui voli, la prenotazione di 
             biglietti aerei e la gestione delle prenotazioni esistenti.



3. API Aviation Stack
        
        L'API AviationStack è un servizio web che fornisce informazioni sul settore dell'aviazione attraverso 
        richieste e risposte basate sul protocollo HTTP. L'API AviationStack offre funzionalità relative all'aviazione,
        tra cui informazioni sui voli, metadati sugli aeroporti, informazioni sugli orari di volo, dati sui 
        codici di compagnie aeree, dati sulle rotte dei voli e altro ancora.
        Nello specifico abbiamo selezionato la seguente funzionalià:

                -dati sui voli: l'API consente di ottenere informazioni in tempo reale sui voli, inclusi i dettagli sullo stato del volo, l'orario di partenza e di arrivo, la rotta del volo, 
                l'aeromobile utilizzato e altro ancora.

4. OpenAI Chat API
        
        L'API offre un modo semplice per interagire con il modello di ChatGPT.
        Nello specifico la utilizziamo affinché possa esserci una chat per l'utente in modo tale da interagire con 
        il bot in quanto l'API permette di definire conversazioni strutturate tra un utente e il modello di ChatGPT.
